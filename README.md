# Smart Doorbell - Face Recognition

## Deployment on Streamlit Community Cloud

1. Upload this project to a GitHub repository.
2. Go to https://share.streamlit.io
3. Connect your GitHub repo.
4. Select:
   - Branch: main
   - File: app.py
5. Click Deploy.

## Folder Structure

- app.py
- requirements.txt
- known_faces/
- unknown_faces/

## Note
Upload your known faces inside the 'known_faces' folder before deployment.
